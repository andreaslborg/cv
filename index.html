A Program that Solves a Problem 1. Semester Project
October, 2020 - November, 2020
The purpose of this project is to develop a software solution that is capable of simulating the OEE of manufacturing processes in order to test organisational changes, thus reducing waste. To perform this simulation, a probabilistic model utilising Monte Carlo simulation and inverse transform sampling is developed. The software solution is successful in simulating the OEE of manufacturing processes.
