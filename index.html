Design, Definition and Implementation of Programming Languages
4. Semester Project
February, 2022 - May, 2022
The proposed solution is a domain specific programming language named Another Card Game Language (ACGL). The constructs of ACGL were implemented based on an analysis of French suited card games, where the fundamental components of card games were explored. It was established, that ACGL must be able to imitate the flow and actions of any arbitrary card game. In order to create a programming language, a compiler was created. The compiler was implemented in Java, using the parser generator ANTLR4. The finished version of ACGL is concluded to be successful in easing the development of French suited card games.
